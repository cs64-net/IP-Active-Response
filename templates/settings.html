API Usage

All endpoints require the X-API-Key header. Base URL: /api/v1

POST/api/v1/block — Block a single IP 
curl -X POST {{ request.host_url }}api/v1/block \
  -H "X-API-Key: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"ip": "10.0.0.1", "note": "Malicious activity"}'
POST/api/v1/block/bulk — Block multiple IPs 
curl -X POST {{ request.host_url }}api/v1/block/bulk \
  -H "X-API-Key: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"ips": ["10.0.0.1", "10.0.0.2"], "note": "Bulk block"}'
POST/api/v1/unblock — Remove an IP from blocklist 
curl -X POST {{ request.host_url }}api/v1/unblock \
  -H "X-API-Key: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"ip": "10.0.0.1"}'

Responses return JSON with success, message, and operation_id fields.
