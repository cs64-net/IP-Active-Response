Select Services

                    

                        
                        This wizard deploys OpenCanary, an open-source honeypot daemon.
                        It runs fake network services (SSH, FTP, HTTP, etc.) on a device in your network. When an attacker connects to one of these decoy services,
                        OpenCanary sends an alert back to this dashboard, and the attacker's IP is automatically blocked across your firewall.
                    

                    
Choose which honeypot services to enable. Each service listens on a port and mimics a real server. You can customise the port for each.

                    

                        Make sure the ports you choose don't conflict with services already running on the target machine. For example, if SSH is already running on port 22, don't enable the SSH honeypot on port 22 — use a different port or disable it.
