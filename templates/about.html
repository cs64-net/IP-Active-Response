IP Active Response

Cyber Membrane Suite

Version 0.1 Beta

Rapid-response IP blocking for SOC teams. Push blocks across your entire network, deploy honeypots that auto-block attackers, and enforce geo-level restrictions — all from one interface.

Network-Wide IP Blocking

Add an IP to the blocklist and push it to every managed device simultaneously. Supports pfSense, Cisco, Fortinet, Palo Alto, Juniper, and more.

Honeypot Auto-Block

Deploy OpenCanary honeypots that report back. Attackers are automatically blocked across the entire network the moment they're detected.

Geo & DNS Blocking

Block entire countries using IP blocklists. Resolve malicious domains and block their IPs on devices that lack native DNS filtering.

Threat Feeds & SIEM

Subscribe to external blocklists with auto-refresh. Forward honeypot alerts to Elasticsearch or Syslog for centralised monitoring.

Quick Start
  1. Add your network devices on the Devices page
  2. Block IPs from the Dashboard — they push to all devices automatically
  3. Set up honeypots on the Honey Pot page — attackers get blocked instantly
  4. Subscribe to threat feeds on External Blocklists for automated protection
